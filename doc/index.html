Tab it!

Just javascript tabs. Make it where you want.
No dependencies. Lightweight. Customizable.

Download or view on GitHub

Install

With npm

$ npm install tabit

With bower

$ bower install tabit

Or download manually.


Include main js file

<script src="path/to/tabit.min.js"></script>

Make HTML

<div class="js-tabs">
  <ul>
    <li>
      <a href="#tab1">Tab 1</a>
    </li>
    <li>
      <a href="#tab2">Tab 2</a>
    </li>
    <li>
      <a href="#tab3">Tab 3</a>
    </li>
  </ul>
  <div id="tab1">Tab content 1</div>
  <div id="tab2">Tab content 2</div>
  <div id="tab3">Tab content 3</div>
</div>

Run Tabit in you document.ready wrapper

var tabs = document.querySelector('.js-tabs');
new Tabit(tabs, options);

Linked by custom attributes

new Tabit(document.getElementById('tabs'), {
  tabSelector: '.js-tab',
  contentSelector: '.js-tab-content',
  tabAttr: 'data-tab',
  contentAttr: 'data-content'
});
<div id="tabs-attrs">
  <div class="tabs__btns">
    <button data-tab="tabs-1" class="tabs__btn js-tab">Tab 1</button>
    <button data-tab="tabs-2" class="tabs__btn js-tab">Tab 2</button>
    <button data-tab="tabs-3" class="tabs__btn js-tab">Tab 3</button>

Options


tabSelector string

Selector inside tabs container for find tab buttons.

Default: a


contentSelector string

Selector inside tabs container for find tab contents.

Default: div


tabAttr string

Tab button's attribute where stored tab content identifier. If use 'href' symbol '#' will be removed.

Default: href


contentAttr string

Tab content's attribute where stored identifier for linking with tab button.

Default: id


tabActiveClass string

Class for active tab button.

Default: is-active


contentActiveClass string

Class for active tab content.

Default: is-active


tabEvent string

Event for toggle active tab.

Default: click


active string

Active tab after init. Set false for no active tab on init.

Default: 1


before function

Callback runs before tab toggled.

Default: null


after function

Callback runs after tab toggled.

Default: null
