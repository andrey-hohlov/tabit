# Tab it!
Just tabs. Make it where you want. No dependencies. Lightweight. Customizable. 


## Install

With npm

```$ npm install tabit```

With bower

```$ bower install tabit```

Download

You can download the latest version or checkout all the releases [here](https://github.com/andrey-hohlov/tabit/releases).
